# Transformer Accounting Gym

A small browser-based drill app for practicing transformer tensor-shape symbols,
decoder-block operations, and path recall.

## Launch

Open `index.html` in a web browser.

On macOS, from this folder:

```sh
open index.html
```

You can also start a simple local server:

```sh
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Project Files

- `index.html` - app markup
- `styles.css` - visual styling
- `app.js` - drill logic
- `assets/transformer-diagram.png` - reference diagram
